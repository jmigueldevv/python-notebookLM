# python-notebookLM
Esse repositório é um pequeno projeto que faz parte do Bootcamp gratuito: “Accenture - Python para Análise e Automação de Dados”, da plataforma DIO. O objetivo principal do projeto é criar um caderno temático no NotebookLM.
O assunto escolhido para meu caderno tematico foi o Python, tendo como objetivo juntar diversas fontes sobre o assunto (EX: videos no youtube, Documentação e etc), facilitando assim o aprendizado, revisão e prática dos conteúdos aprendidos em Python.
